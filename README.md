# KODECAMP NODE TASK

## DESCRIPTION

Kodecamp node js CRUD api task

## INSTRUCTIONS

```bash
git clone [link](https://github.com/billmal071/kodekamp-nodejs-task)
npm i
npm run dev
```

## ROUTES

```bash
  GET /info?pages=1&pageSize=10</>
  GET /info/:id
  POST /info
  PUT /info/:id
  PATCH /info/:id
  DELETE /info/:id
```
